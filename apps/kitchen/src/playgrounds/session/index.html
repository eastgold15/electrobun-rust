Session & Partition Playground

Test webview partitions and session isolation with localStorage

How It Works

Each webview below has a counter stored in localStorage.

Webviews with the same partition share localStorage and will show the same counter value.

Webviews with different partitions have isolated storage and independent counters.

Try it: Click +/- in one webview and watch which others update!
